# Theoria Books Agent Harness

This directory is the persistent AI harness for Theoria Books. It exists so clean-session agents can resume work without rediscovering the project rules from chat history.

Mandatory order:

1. Read `COLDSTART.md`.
2. Read `ARCHITECTURE.md`.
3. Read `QUALITY_RULES.md`.
4. Read `DESIGN_RULES.md`.
5. Read `VERIFICATION.md`.
6. Only then inspect or edit source files.

The app lives in `frontend/`.
