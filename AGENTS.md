# Agent Bootstrap

Before making changes in this repository, every coding agent must read and follow:

1. `.agents/COLDSTART.md`
2. `.agents/ARCHITECTURE.md`
3. `.agents/QUALITY_RULES.md`
4. `.agents/DESIGN_RULES.md`
5. `.agents/VERIFICATION.md`

These files are mandatory project instructions. If a request conflicts with them, preserve the product architecture and quality rules unless the user explicitly asks to change the rules themselves.
